#  Uber Clone App (MERN + Flask Full Stack)

A full-stack user management application built using MERN and Flask, featuring secure authentication, API documentation with Swagger, and scalable architecture for real-world applications.

---

## Features

- JWT-based Authentication (Register/Login)
-  User Management (CRUD)
-  Swagger API Documentation
-  Request Validation (express-validator / Flask validation)
-  Secure Password Hashing (bcrypt)
-  RESTful API Architecture
- Full Stack Integration (Frontend + Backend)
-  Modular & Scalable Structure

---

##  Tech Stack

### Frontend
- React.js / Vue.js (depending on implementation)
- Axios
- Bootstrap / Tailwind CSS

### Backend
- Node.js
- Express.js
- Flask (for additional services / APIs)

### Database
- MongoDB

### Other Tools
- JWT (Authentication)
- Swagger (API Docs)
- express-validator
- bcryptjs
- dotenv
